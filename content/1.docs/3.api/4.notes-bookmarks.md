## Vue d'ensemble

Les endpoints Notes et Signets permettent aux utilisateurs de sauvegarder et gérer leurs références bibliques favorites avec un système de couleurs, et de créer des annotations personnelles sur les versets.

## Signets

### Liste des signets utilisateur
Récupérer tous les signets de l'utilisateur connecté avec pagination et filtres.

::code-group
```bash [GET]
GET /api/bible/bookmarks

# Auth: Requise

# Paramètres de query
color: Filtrer par couleur spécifique (optionnel)
limit: Nombre maximum de résultats (optionnel, défaut: 20)
offset: Décalage pour pagination (optionnel, défaut: 0)
book: Filtrer par livre biblique (optionnel)
version: Filtrer par version (optionnel)
```

```json [Response]
{
  "success": true,
  "data": {
    "bookmarks": [
      {
        "id": 1,
        "title": "Mon verset préféré",
        "color": "blue",
        "reference": "Jean 3:16",
        "verse": {
          "id": 26114,
          "text": "Car Dieu a tant aimé le monde qu'il a donné son Fils unique...",
          "chapter": 3,
          "verse": 16,
          "version": {
            "code": "LSG",
            "name": "Louis Segond 1910"
          }
        },
        "book": {
          "code": "JHN",
          "name": "Jean",
          "testament": "Nouveau Testament"
        },
        "createdAt": "2025-09-29T10:30:00Z",
        "updatedAt": "2025-09-29T10:30:00Z"
      }
      // ... autres signets
    ],
    "pagination": {
      "total": 15,
      "limit": 20,
      "offset": 0,

    },
    "groupedByColor": {

    },
    "colorLabels": {
      "red": "Important",
      "blue": "Méditation",
      "green": "Promesses"
      // ... autres couleurs
    }
  }
}
```

### Créer un nouveau signet
Ajouter un signet coloré sur un verset spécifique.

::code-group
```bash [POST]
POST /api/bible/bookmarks

# Auth: Requise
```

```json [Request Body]
{
  "verseId": 26114,
  "color": "blue",
  "title": "Mon verset préféré"
}
```

```json [Response]
{
  "success": true,
  "data": {
    "bookmark": {
      "id": 1,
      "title": "Mon verset préféré",
      "color": "blue",
      "reference": "Jean 3:16",
      "verse": {
        "id": 26114,
        "text": "Car Dieu a tant aimé le monde...",
        "chapter": 3,
        "verse": 16
      },
      "createdAt": "2025-09-29T15:22:00Z"
    },
    "message": "Signet créé avec succès"
  }
}
```
::

### Supprimer un signet
Supprimer définitivement un signet existant.

::code-group
```bash [DELETE]
DELETE /api/bible/bookmarks/{id}

# Auth: Requise
# Paramètres de l'URL
id: ID numérique du signet à supprimer
```

```json [Response]
{
  "success": true,
  "message": "Signet supprimé avec succès"
}
```
::

## Notes personnelles

### Liste des notes utilisateur
Récupérer toutes les notes de l'utilisateur avec filtres et pagination.

::code-group
```bash [GET]
GET /api/bible/notes

# Auth: Requise

# Paramètres de query
book: Filtrer par livre biblique (optionnel)
private: Filtrer par confidentialité "true"/"false" (optionnel)
limit: Nombre maximum de résultats (optionnel, défaut: 10)
offset: Décalage pour pagination (optionnel, défaut: 0)
search: Recherche dans titre et contenu (optionnel)
```

```json [Response]
{
  "success": true,
  "data": {
    "notes": [
      {
        "id": 1,
        "title": "Réflexion sur l'amour de Dieu",
        "content": "Ce verset révèle la profondeur de l'amour divin...",
        "isPrivate": false,
        "verse": {
          "id": 26114,
          "text": "Car Dieu a tant aimé le monde...",
          "chapter": 3,
          "verse": 16,
          "version": {
            "code": "LSG",
            "name": "Louis Segond 1910"
          }
        },
        "book": {
          "code": "JHN",
          "name": "Jean",
          "testament": "Nouveau Testament"
        },
        "createdAt": "2025-09-29T10:15:00Z",

      // ... autres notes
```

### Créer une nouvelle note
Ajouter une note personnelle sur un verset biblique.

::code-group
```bash [POST]
POST /api/bible/notes

# Auth: Requise
```

```json [Request Body]
{
  "verseId": 26114,
  "title": "Réflexion sur l'amour de Dieu",
  "content": "Ce verset révèle la profondeur de l'amour divin pour l'humanité...",
  "isPrivate": false
}
```

```json [Response]
{
  "success": true,
  "data": {
    "note": {
      "id": 1,
      "title": "Réflexion sur l'amour de Dieu",
      "content": "Ce verset révèle la profondeur...",
      "isPrivate": false,
      "verse": {
        "id": 26114,
        "text": "Car Dieu a tant aimé le monde...",
        "chapter": 3,
        "verse": 16
      },
      "createdAt": "2025-09-29T15:30:00Z"
    },
    "message": "Note créée avec succès"
  }
}
```
::

### Modifier une note existante
Mettre à jour le contenu d'une note personnelle.

::code-group
```bash [PUT]
PUT /api/bible/notes/{id}

# Auth: Requise
# Paramètres de l'URL
id: ID numérique de la note à modifier
```

```json [Request Body]
{
  "title": "Nouveau titre de la note",
  "content": "Contenu mis à jour de la note...",
  "isPrivate": true
}
```

```json [Response]
{
  "success": true,
  "data": {
    "note": {
      "id": 1,
      "title": "Nouveau titre de la note",
      "content": "Contenu mis à jour...",
      "isPrivate": true,
      "updatedAt": "2025-09-29T16:45:00Z"
    },
    "message": "Note mise à jour avec succès"
  }
}
```
::

## Notes

### Liste des notes
Récupérer toutes les notes de l'utilisateur.

::code-group
```bash [GET]
GET /api/bible/notes

# Paramètres de query
reference: Référence biblique pour filtrer (optionnel)
version: Code de la version biblique (optionnel)
limit: Nombre maximum de résultats (optionnel, par défaut: 20)
offset: Nombre de résultats à sauter (optionnel, par défaut: 0)
```

```json [Response]
{
  "notes": [
    {
      "id": "uuid",
      "reference": "Jean 3:16",
      "content": "Une note importante sur ce verset...",
      "createdAt": "2025-09-23T10:30:00Z",
      "updatedAt": "2025-09-23T10:30:00Z"
    },
    // ...autres notes
  ]
}
```
::
