## Vue d'ensemble

L'API Bible fournit des endpoints pour accéder aux textes bibliques, gérer les versions disponibles, effectuer des recherches, et manipuler les notes et signets personnels.

## Livres et Versets

### Liste des livres bibliques
Récupérer la liste complète des livres bibliques avec leurs métadonnées.

::code-group
```bash [GET]
GET /api/bible/books
```

```json [Response]
{
  "success": true,
  "data": {
    "all": [
      {
        "id": 1,
        "code": "GEN",
        "name": "Genèse",
        "testament": "Ancien Testament",
        "chapters": 50,
        "order": 1
      },
      {
        "id": 2,
        "code": "EXO", 
        "name": "Exode",
        "testament": "Ancien Testament",
        "chapters": 40,
        "order": 2
      }
      // ... autres livres
    ],
    "oldTestament": [...],
    "newTestament": [...]
  }
}
```
::

### Versets d'un chapitre complet
Récupérer tous les versets d'un chapitre spécifique.

::code-group
```bash [GET]
GET /api/bible/verses/{livre}/{chapitre}

# Paramètres de l'URL
livre: Code du livre biblique (ex: GEN, JHN, PSA)
chapitre: Numéro du chapitre

# Paramètres de query
version: Code de la version (optionnel, défaut: LSG)
```

```json [Response]
{
  "success": true,
  "data": {
    "verses": [
      {
        "id": 1,
        "verse": 1,
        "text": "Au commencement, Dieu créa les cieux et la terre.",
        "chapter": 1,
        "book": {
          "code": "GEN",
          "name": "Genèse"
        },
        "version": {
          "code": "LSG",
          "name": "Louis Segond 1910"
        }
      }
      // ... autres versets du chapitre
    ],
    "book": {
      "code": "GEN",
      "name": "Genèse",
      "testament": "Ancien Testament"
    },
    "chapter": 1,
    "version": "LSG"
  }
}
```
::

### Verset spécifique
Récupérer un verset précis avec ses métadonnées complètes.

::code-group
```bash [GET]
GET /api/bible/verses/{livre}/{chapitre}/{verset}

# Paramètres de l'URL
livre: Code du livre biblique (ex: JHN, PSA, ROM)
chapitre: Numéro du chapitre
verset: Numéro du verset

# Paramètres de query
version: Code de la version (optionnel, défaut: LSG)
```

```json [Response]
{
  "success": true,
  "data": {
    "verse": {
      "id": 26114,
      "verse": 16,
      "text": "Car Dieu a tant aimé le monde qu'il a donné son Fils unique, afin que quiconque croit en lui ne périsse point, mais qu'il ait la vie éternelle.",
      "chapter": 3,
      "book": {
        "id": 43,
        "code": "JHN", 
        "name": "Jean",
        "testament": "Nouveau Testament"
      },
      "version": {
        "id": 1,
        "code": "LSG",
        "name": "Louis Segond 1910"
      }
    },
    "reference": "Jean 3:16",
    "version": "LSG"
  }
}
```
::

## Recherche

### Recherche textuelle
Effectuer une recherche dans l'ensemble des Écritures.

::code-group
```bash [GET]
GET /api/bible/search

# Paramètres de query (requis)
q: Terme de recherche
version: Code de la version (optionnel, défaut: LSG)
limit: Nombre de résultats max (optionnel, défaut: 50)
offset: Décalage pour pagination (optionnel, défaut: 0)
book: Filtrer par livre spécifique (optionnel)
```

```json [Response]
{
  "success": true,
  "data": {
    "verses": [
      {
        "id": 26114,
        "verse": 16,
        "text": "Car Dieu a tant aimé le monde qu'il a donné son Fils unique...",
        "chapter": 3,
        "book": {
          "code": "JHN",
          "name": "Jean"
        },
        "version": {
          "code": "LSG",
          "name": "Louis Segond 1910"
        }
      }
      // ... autres résultats
    ],
    "pagination": {
      "total": 127,
      "limit": 50,
      "offset": 0,
      "hasMore": true
    },
    "query": "amour",
    "version": "LSG"
  }
}
```
::

## Comparaison de versions

### Comparer des versions
Comparer le même passage dans différentes versions bibliques.

::code-group
```bash [POST]
POST /api/bible/compare

# Auth: Requise
```

```json [Request Body]
{
  "book": "JHN",
  "chapter": 3,
  "verse": 16,
  "versions": ["LSG", "S21", "NEG"]
}
```

```json [Response]
{
  "success": true,
  "data": {
    "reference": "Jean 3:16",
    "comparisons": [
      {
        "version": {
          "code": "LSG",
          "name": "Louis Segond 1910"
        },
        "text": "Car Dieu a tant aimé le monde qu'il a donné son Fils unique..."
      },
      {
        "version": {
          "code": "S21", 
          "name": "Segond 21"
        },
        "text": "En effet, Dieu a tant aimé le monde qu'il a donné son Fils unique..."
      }
      // ... autres versions
    ]
  }
}

# Paramètres de query
version: Code de la version biblique (optionnel, par défaut: version préférée de l'utilisateur)
```

```json [Response]
{
  "book": "GEN",
  "chapter": 1,
  "verses": [
    {
      "number": 1,
      "text": "Au commencement, Dieu créa les cieux et la terre."
    },
    // ...autres versets
  ]
}
```
::

## Versions et Comparaison

### Liste des versions
Obtenir les versions bibliques disponibles.

::code-group
```bash [GET]
GET /api/bible/versions

# Paramètres de query
language: Code de langue (optionnel, par défaut: fr)
```

```json [Response]
{
  "versions": [
    {
      "id": "LSG",
      "name": "Louis Segond 1910",
      "language": "fr"
    },
    // ...autres versions
  ]
}
```
::

### Comparaison de versions
Comparer des versets entre différentes versions.

::code-group
```bash [POST]
POST /api/bible/compare
```

```json [Body]
{
  "reference": "Jean 3:16",
  "versions": ["LSG", "NEG1979"]
}
```

```json [Response]
{
  "reference": "Jean 3:16",
  "comparisons": [
    {
      "version": "LSG",
      "text": "Car Dieu a tant aimé le monde..."
    },
    {
      "version": "NEG1979",
      "text": "Car Dieu a tant aimé le monde..."
    }
  ]
}
```
::

## Préférences de lecture

### Obtenir les préférences
Récupérer les préférences de lecture de l'utilisateur.

::code-group
```bash [GET]
GET /api/bible/preferences
```

```json [Response]
{
  "defaultVersion": "LSG",
  "fontSize": 16,
  "theme": "light"
}
```
::

### Mettre à jour les préférences
Modifier les préférences de lecture.

::code-group
```bash [PUT]
PUT /api/bible/preferences
```

```json [Body]
{
  "defaultVersion": "NEG1979",
  "fontSize": 18,
  "theme": "dark"
}
```
::

::tip
Les préférences sont automatiquement sauvegardées et synchronisées entre les appareils.
